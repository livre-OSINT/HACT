HACT vs MITRE ATT&CK®

ATT&CK describes technical actions against systems; HACT describes cognitive maneuvers against decisions. Same two-level structure (tactics → techniques), different object.

Architecture
CVE → HVE mapping
Persuasion principles

HVE (Human Vulnerabilities and Exploits) are the human equivalent of CVEs: not software bugs but exploitable cognitive biases and psychological needs. A CVE is patched; an HVE is compensated for with structured vigilance.

Detection — Indicators of HUMINT Operation (IoH)
